# APP-Sorteio
## Criando primeiro App no **AndroidStudio**
### O App Sorteio básicamente sorteia um número de 0 a 10, clicando no botão [JOGAR].
### Ao clicar no botão [CLIQUE AQUI!], aparece uma mensagem de agradecimento.
